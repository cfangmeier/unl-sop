\section{Equipment}

\begin{description}
\item[Probe station] to hold the setup
\item[Pico probe model 7] mounted on microspositioner, used for GND connection
\item[Pico probe model 7A] mounted on micropositioner, used for HV connection
\item[Probe tips] Model 7-125, one for each pico probe
\item[Computer] Requires pXar installed
\item[DTB] Connected to the computer through a USB cable. No need to connect anything else, DTB just required for pXar to work properly.
\item[HV power supply] use Keithly model 2430, connected to the same computer through a USB to RS232 adapter
\item[Connection box] to make proper connection between picoprobes and HV power supply
\end{description}
